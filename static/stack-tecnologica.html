Quadro 12 – Stack Tecnológica — CondConnect

Camada Tecnologia Finalidade
Apresentação HTML5 Estruturação semântica das páginas.
Apresentação CSS3 com variáveis customizadas Estilização visual e padronização de tema.
Apresentação JavaScript ES6+ (Vanilla JS) Interatividade e consumo da API REST via fetch API.
Apresentação Google Fonts (Outfit) Tipografia da interface.
Apresentação SVG inline Renderização de ícones sem dependência externa.
Lógica de negócio Python 3 Linguagem do back-end.
Lógica de negócio Flask Microframework Web para API REST.
Lógica de negócio Flask-Login Autenticação com sessões server-side e cookies seguros.
Lógica de negócio Werkzeug Hash de senhas e utilitários de upload.
Lógica de negócio Flask-CORS Controle de origem das requisições HTTP.
Lógica de negócio Gunicorn Servidor WSGI de aplicação Python.
Lógica de negócio Apache HTTP Server Proxy reverso e exposição pública do back-end.
Persistência MySQL Banco de dados relacional principal.
Persistência PyMySQL (DictCursor) Driver de conexão Python ↔ MySQL com autocommit habilitado. Resultados retornados como dicionários.
Persistência Sistema de arquivos local Armazenamento das imagens dos anúncios em /static/assets/uploads/, com nomes únicos gerados por UUID.
Infraestrutura AWS Lightsail (Ubuntu) Hospedagem da aplicação em nuvem com instância Linux Ubuntu.
Infraestrutura Git Controle de versão e deploy via git pull diretamente na instância do servidor.
