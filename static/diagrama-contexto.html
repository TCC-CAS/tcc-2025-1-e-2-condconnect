CondConnect
Marketplace exclusivo para condomínios residenciais. Backend Python/Flask + Frontend HTML/CSS/JS.

Hospedado e executado na instância Linux
Grava e serve imagens dos produtos (upload)
«system»
Banco de Dados Relacional (MySQL)
Armazena dados de usuários, produtos, pedidos, avaliações, notificações e logs. Hospedado em servidor externo.
«external system»
AWS Lightsail (Servidor Linux/Ubuntu)
Infraestrutura de hospedagem. Executa Gunicorn + Apache como proxy reverso. Deploy via Git.
